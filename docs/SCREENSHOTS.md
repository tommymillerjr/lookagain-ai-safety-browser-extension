# Screenshots

These screenshots show the public portfolio version of LookAgain / BrowseGuard AI.

Screenshots using PayPal-like wording are from a local simulated phishing test page running on localhost. They are not screenshots of a live PayPal site.

## 01-safe-page-popup.png

Shows a low-risk page result with minimal interruption. This demonstrates alert-fatigue restraint and normal browsing behavior when no significant risk signals are detected.

## 02-high-risk-warning-overlay.png

Shows a high-risk warning on a local simulated phishing page. This demonstrates user-facing intervention, explainable risk signals, and user choice.

## 03-high-risk-page-report.png

Shows the generated page report for the simulated high-risk page. This demonstrates risk scoring, matched signals, reasons, trust state, local protection status, and privacy-aware reporting.

## 04-privacy-first-ai-settings.png

Shows the settings dashboard with local heuristics, mock AI advisory behavior, and provider calls disabled by default. This demonstrates privacy-first safety design.
